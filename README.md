# Simple REST Bank Card Management System

## About project
This project is a simple backend application for managing bank cards, including encryption of card data, encryption of user passwords, authentication via a JWT token, as well as basic operations on the card and user account.

## Features

**User**:
- Registers new account or log in into existing one
- Views their balance
- Vies their cards
- Makes transfers between their cards
- Requests a card to be blocked

**Admin**:
- Manages users
- Blocks, Activates and Deletes cards
- Views all cards

## Technologies
- **Java 21**
- **Spring**: Spring boot, Spring Security, Spring Data JPA
- **Database**: PostgreSQL 17
- **Migrations**: Liquibase
- **Docker**
- **JWT**
- **Swagger (OpenAPI)**
